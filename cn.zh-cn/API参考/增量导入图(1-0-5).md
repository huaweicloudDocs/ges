# 增量导入图\(1.0.5\)<a name="ges_03_0073"></a>

## 功能介绍<a name="section11581657646"></a>

增量导入图数据。

>![](public_sys-resources/icon-note.gif) **说明：**   
>为防止系统重启时，不能正常恢复导入图数据，建议在使用图期间，不要删除存储在OBS中的数据。
