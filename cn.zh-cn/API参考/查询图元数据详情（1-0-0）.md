## 请求<a name="section3206722619349"></a>

-   请求样例

    ```
    GET http://{SERVER_URL}/ges/v1.0/{projectId}/graphs/{graphName}/schema
    ```

    >![](public_sys-resources/icon-note.gif) **说明：** 
    >SERVER\_URL：图的访问地址，取值请参考[业务面API使用限制](业务面API使用限制.md)。
