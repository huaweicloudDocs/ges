# 查询节点详情<a name="ges_03_0025"></a>

## 功能介绍<a name="section1856650418852"></a>

根据节点ID查询节点信息，返回节点详情，包括标签和属性等。

## URI<a name="section4609985518852"></a>

-   URI 格式

    ```
    GET /ges/v1.0/{projectId}/graphs/{graphName}/vertices/detail?vertexIds={vertexIds}
    ```
