    ```
    Http Status Code: 200
    {   "quotas": {
           "resources": [
               {
                   "type": "graph",
                   "available": 1,
                   "edgeVolume": 178800
               },
               {
                   "type": "backup",
                   "available": 7,
                   "edgeVolume": 0
               }
           ]
       }
     }
    ```

-   请求失败样例

    ```
    Http Status: 400
    {
       "errorCode": "GES.7006",
       "errorMessage": "The underlying graph engine has internal error."
    }
    ```
