# 批量删除边\(2.1.9\)<a name="ges_03_0155"></a>

## 功能介绍<a name="section37889131194723"></a>

根据批量边的起点、终点以及索引，删除这些边。

## URI<a name="section39433874194723"></a>

-   URI格式

    ```
    POST /ges/v1.0/{projectId}/graphs/{graphName}/edges/action?action_id=batch-delete
    ```
