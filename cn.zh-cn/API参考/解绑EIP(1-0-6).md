# 解绑EIP\(1.0.6\)<a name="ges_03_0104"></a>

## 功能介绍<a name="section169600301490"></a>

当无需继续使用EIP时，您可通过解绑EIP来释放网络资源。

## URI<a name="section149741230134910"></a>

    ```
    POST  /v1.0/{projectId}/graphs/{graphId}/action?action_id=unbindEip
    ```
