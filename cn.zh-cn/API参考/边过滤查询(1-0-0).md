## 请求<a name="section24693948175023"></a>

-   请求样例

    ```
    POST http://{SERVER_URL}/ges/v1.0/{projectId}/graphs/{graphName}/edges/action?action_id=query
    {
        "offset":"0",
        "limit":"20",
        "labels":[
            "rate"
        ],
        "edgeFilters":[
            {
                "propertyName":"Score",
                "predicate":">=",
                "values":[
                    "2"
                ]
            },
            {
                "propertyName":"Datetime",
                "predicate":"range",
                "values":[
                    "1998-12-27 01:00:00",
                    "2000-12-31 00:12:38"
                ],
                "type":"or"
            }
        ]
    }
    ```
