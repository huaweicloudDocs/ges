# 升级图\(1.0.5\)<a name="ges_03_0101"></a>

## 功能介绍<a name="section169600301490"></a>

升级图。图引擎服务会定期升级版本，用户可根据需要升级图。

## URI<a name="section149741230134910"></a>

-   URI 格式

    ```
    POST  /v1.0/{projectId}/graphs/{graphId}/action?action_id=upgrade
    ```
