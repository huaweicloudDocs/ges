## 功能介绍<a name="section5422958920145"></a>

根据给定的 indexName 删除索引。

## URI<a name="section624997620145"></a>

-   URI 格式

    ```
    DELETE /ges/v1.0/{projectId}/graphs/{graphName}/indices/{indexName}
    ```
